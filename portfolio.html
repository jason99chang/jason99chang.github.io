Kickstarter Analytics

Kickstarter Analytics is a webapp that can search, create, read, update, and delete over three hundred thousand entries on kickstarter data. It was created using bootstrap and flask. The webapp can also display graphs to display the most up to date information on the data.

RShell

RShell is a basic command shell that I created with a partner. The project used a fork function to handle multiple commands at once. We implemented different design patterns to help interpret requests that took multiple commands at once using c++.

LoL Data Project

The LoL data project uses the Riot Games API and Pandas framework to analyze data. We used this data to make predictions using linear regression techniques. We then presented our findings on a website that used the React framework.

Calculator

I created a calculator that uses nine different design patterns in c++. This is so that the calculator is able to handle multiple functoins at once in the correct order. We used google tests for testing all of the different functions.
